# XSS Payloads Collection & Modern Cross-Site Scripting Research

Repositorio enfocado en el estudio de vulnerabilidades Cross-Site Scripting (XSS), incluyendo payloads clásicos, modernos y técnicas utilizadas actualmente en pentesting, bug bounty y análisis de aplicaciones web modernas.

Este proyecto reúne:
- payloads históricos,
- payloads modernos,
- bypass de filtros,
- técnicas de evasión CSP,
- DOM XSS,
- Mutation XSS,
- SVG XSS,
- Markdown XSS,
- y ejemplos prácticos utilizados en entornos reales.

---

## Contenido

- XSS clásico
- XSS moderno
- DOM Based XSS
- Stored XSS
- Reflected XSS
- SVG XSS
- Markdown XSS
- CSP Bypass
- Mutation XSS (mXSS)
- Polyglot Payloads
- Blind XSS
- DOM Clobbering
- Payloads para frameworks modernos
- Técnicas de evasión WAF
- Payloads legacy e históricos

---

## Objetivo

El objetivo de este repositorio es:
- documentar técnicas XSS,
- entender cómo funcionan los payloads,
- aprender evasión de filtros,
- estudiar parsing HTML/DOM,
- practicar pentesting web,
- y servir como referencia educativa y de investigación.
